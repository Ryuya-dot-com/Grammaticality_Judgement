Grammaticality Judgment Task

Reading-While-Listening (Web Pilot)

詳細設定

使い方: 各試行で画面に表示される文を読み、音声を聞きながら、 その文がその文脈で文法的に正しいかどうかを判断してください。
1 = 文法的に正しい / 2 = 正しくない
（またはボタンをクリック）

実験の進め方

各試行は次のように進みます:

  1. 場面を説明する文 (プライム) が表示され、同時に音声が流れます。
  2. 短い間。
  3. 判断対象の文 (ターゲット) が表示され、同時に音声が流れます。
  4. 反応してください: 1 = 文法的 / 2 = 非文法的

全 4 ラン（各 ~5 分）の実験です。各ランの開始時、「Begin Run」ボタンを押してください。

⚠️ ブラウザの自動再生ポリシーのため、最初の試行は音声が流れない場合があります。 「Begin Run」を押すことでオーディオが有効化されます。
